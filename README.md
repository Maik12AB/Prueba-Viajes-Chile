# ✈️ Viajes Chile

Proyecto desarrollado como parte de un desafío del Bootcamp de Desarrollo Full Stack.

## 📖 Descripción

Landing page responsiva para una agencia de viajes ficticia llamada **Viajes Chile**. El proyecto fue desarrollado utilizando HTML, CSS, Bootstrap y JavaScript, poniendo en práctica conceptos como diseño responsivo, componentes de Bootstrap y manipulación básica del DOM.

## 🚀 Demo

🔗 https://maik12ab.github.io/Prueba-Viajes-Chile/


## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts

## ✨ Funcionalidades

- Navbar fija con navegación entre secciones.
- Carrusel de imágenes.
- Sección "¿Quiénes Somos?".
- Tarjetas con destinos destacados.
- Formulario de contacto.
- Diseño completamente responsivo.
- Tooltips e interacción mediante JavaScript.

## 🎯 Objetivo

Aplicar los conocimientos adquiridos durante el bootcamp, utilizando Bootstrap para construir una página moderna, adaptable a distintos dispositivos y con una experiencia de usuario agradable.
